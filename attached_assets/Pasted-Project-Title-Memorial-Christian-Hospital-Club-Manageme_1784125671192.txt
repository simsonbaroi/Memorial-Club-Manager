Project Title

Memorial Christian Hospital Club Management System (MCHCMS)

Project Overview

Develop a complete Enterprise Resource Planning (ERP) Progressive Web Application (PWA) for Memorial Christian Hospital Club, Malumghat, Chakaria, Cox's Bazar, Bangladesh.

This is a non-profit club. The application will manage every financial and administrative activity of the club for many years.

The system must be designed to run 100% free, without depending on paid APIs or paid cloud services.

The application must be offline-first, support automatic synchronization between multiple devices, and work seamlessly on Windows, Android, iPhone, macOS, and Linux.

The application should be professionally designed and scalable for at least the next 20–30 years without changing the database structure.

Final Instruction to Replit AI

Do not generate this project in a single step.

First, create a complete project architecture and folder structure.
Then create the SQLite database schema and migrations.
Then implement authentication.
Then build each module incrementally.
After each completed module, verify functionality before proceeding to the next.
Use clean, documented code with comments, reusable components, and maintainable architecture. Do not use placeholder implementations where a complete implementation is feasible.
